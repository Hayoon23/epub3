실습과제
루프 반복문 활용

학습 목표

1. 루프의 기능을 이해하고 설명할 수 있다.

2. 사용자 입력 변수를 이해하고 활용할 수 있다.

3. 루프를 이용하여 원하는 만큼 반복 동작을 구현할 수 있다.

1. 부품 및 장비

2. 구현 동작 목표 (7장 실습 과제와 유사)

학습 목표

<그림 XI-1>을 참조하여, 사용자 입력 횟수만큼 픽앤플레이스 작업 후 맨눈 검사를 하고, 검사 결과에 따라서 적절히 처리하는 프로그램을 루프 구조문을 활용하여 구현한다.

동작 설명

① 프로그램 실행과 함께 사용자 입력 팝업이 뜨고, 맨눈 검사를 할 반복 구간을 ‘정수’로 사용자가 직접 입력한다. 단, 사용자 입력 팝업은 프로그램 실행 후 딱 한 번만 생성되도록 한다.

② 맨눈 검사 구간이 아닌 경우 로봇은 지정된 위치로 블록을 옮긴다.

③ 사용자 입력한 반복 순서가 되면 맨눈 검사 모드로 동작한다. 이때 사용자는 스위치를 사용하여 불량 여부 정보를 입력한다.

④ 불량이 아닌 경우 로봇은 지정된 위치로 블록을 옮긴다.

⑤ 불량일 때 로봇은 제품 폐기 위치에 블록을 버린다.
