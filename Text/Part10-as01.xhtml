4. 폴더와 서브프로그램의 차이점을 알고 적절하게 활용할 수 있다

2. 구현 동작 목표

학습 목표

(7장 실습 과제와 동일) 스위치의 상태에 따라서 맨눈 검사 진행 여부 및 검사 결과에 따라 동작을 다르게 하는 동작을 구현한다. 다만, 반복되는 부분을 확인하고 폴더 혹은 서브프로그램을 적절히 활용하여 재작성한다.

동작 설명

<그림 Ⅹ-1> 참조

① 스위치 1(SW1)은 맨눈 검사 여부를, 스위치 2(SW2)는 불량 여부 정보를 사용자가 직접 로봇에게 제공한다.

② (SW1, SW2는 Low 상태; 기본 동작) 로봇은 기본적으로 컨베이어벨트 우측 센서 위치에서 블록을 집고, 컨베이어벨트 좌측으로 블록을 옮겨 놓는 동작을 반복한다. 이 동작은 3장 및 4장에서 작성하였던 프로그램과 같다.

③ 블록을 든 상태에서 맨눈 검사를 할지를 SW1을 통하여 결정한다. SW1이 High이면 맨눈 검사 위치로, Low이면 다시 블록을 기본 동작인 컨베이어벨트 좌측으로 이동시킨다. 맨눈 검사는 블록을 검사자 눈앞으로 먼저 이동시켜 회전함으로써 편하게 관찰할 수 있도록 한다. 회전이 끝나면 2~5초 사이의 대기시간이 주고, 검사자는 SW2를 이용하여 불량 여부에 대한 정보를 로봇에게 제공한다.

④ 불량이 아닌 경우 사진과 같이 기본 동작인 컨베이어벨트 좌측 위치에 이동시킨다.

⑤ 불량일 때는 사진과 같이 사용자가 지정한 위치에 버리는 동작을 구현한다.
